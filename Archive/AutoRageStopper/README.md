# AutoRageStopper

> **Archived:** This experimental mod is no longer maintained and is retained only for reference.

AutoRageStopper was an early utility for ending Rage Mode around time-sensitive world objects.

## Historical Features

- Monitored Rage Mode during normal runner and Rage gameplay states.
- Detected Special Random Boxes and Chest Hunt Keys.
- Ended Rage after a short delay when a protected object was detected.
- Applied a maximum Rage duration as a fallback.
- Allowed Rage to be ended manually.

## Historical Control

- `J`: End the current Rage Mode immediately.

## Status

This project is not actively maintained and may no longer match the current game API. It should not be treated as a supported dependency of AutoProgression or AutoClimber.

## Building

This archived project targets .NET 6 and expects the Idle Slayer Mod Manager/MelonLoader game assemblies. Building it does not imply current compatibility.
