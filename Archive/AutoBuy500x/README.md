# AutoBuy500x

> **Archived:** This experimental mod is no longer maintained. Its useful functionality has been reorganized and expanded in AutoProgression.

AutoBuy500x was an early automation experiment for paid 500x bonuses and Rage Pill handling in Idle Slayer.

## Historical Features

- Purchased the Souls and CPS 500x Jewels of Soul bonuses on fixed timers.
- Checked and crafted Rage Pills.
- Could purchase Root and Slime materials for Rage Pills.
- Avoided Rage Pill actions while a Special Random Box or Chest Hunt Key was present.

## Historical Controls

- `L`: Toggle automatic 500x purchases and Rage Pill crafting.
- `K`: Toggle material refilling and Rage activation after crafting a pill.

## Status

Use AutoProgression for current implementations of paid bonuses, Rage Pills, material purchases, and other progression automation. This project is retained only as historical source and may no longer match the current game API.

## Building

This archived project targets .NET 6 and expects the Idle Slayer Mod Manager/MelonLoader game assemblies. Building it does not imply current compatibility.
