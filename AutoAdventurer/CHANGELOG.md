# Changelog

## 2.0.0

- Unified user-facing logs around concise actions, results, warnings, and
  errors without a special user marker.
- Grouped diagnostics into Config, Runtime, Movement, Rage, Quest, Element,
  SilverBox, Boss, and Gameplay categories under the main Debug Mode.
- Added duplicate-message suppression for noisy diagnostics, warnings, and
  errors, including a suppressed-repeat count when output resumes.
- Improved quest travel, character switching, completion, and session-summary
  messages with stable internal IDs and explicit fields.
- Updated the complete manual, user guide, configuration reference, logging
  reference, and troubleshooting guidance for the current behavior.
- Synchronized assembly, plugin, and Mod Manager metadata at version 2.0.0.
